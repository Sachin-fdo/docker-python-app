# Dockerized Python Application

## Overview
This project demonstrates containerizing a Python application using Docker.  
The app prints system and Python environment information and runs consistently inside a container.

## Technologies
- Python 3.10
- Docker
- Linux
- Git & GitHub

## Steps Performed
1. Developed a Python application (`app.py`)
2. Created `Dockerfile` to containerize the app
3. Built Docker image locally and verified output
4. Pushed Docker image to Docker Hub
5. Uploaded screenshots and documentation to GitHub
